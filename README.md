# Educational Content Platform — Database Design

This repository contains the complete database design for an **Educational Content Presentation Platform**.
The system supports managing user accounts, subscriptions, courses, enrollments, learning progress, assessments, certificates, and university-level administration.

The project includes:
- Full requirements analysis and business rules
- ER diagram design
- ER-to-relational schema mapping
- Normalization (1NF, 2NF, 3NF)
- SQL implementation (table creation + constraints)
- Triggers for automation and auditing
- Sample queries and views for reporting

## Key Features Covered in the Design
- User accounts with subscription status and linked addresses
- Subscription plans with date validation rules
- Course management including levels, pricing, and coordinators
- Enrollment tracking with payment status enforcement
- Student progress tracking per course
- Assessment structure with score validations
- Certificate issuance and signature storage
- Teacher earnings calculation based on course engagement
- Change logging (audit trail) using triggers
